# Швидкий старт - MQL 2004 I2C Encoder V2

## Мінімальне підключення

### Обов'язкові компоненти:
1. **LCD 2004 I2C** → SDA/SCL (A4/A5 для UNO)
2. **Енкодер KY-040** → D2, D3, A3
3. **Кнопка START** → A1
4. **LED START** → A2
5. **Потенціометр** → A0
6. **DM556** → D10, D11, D12

## Перший запуск

1. Завантажте скетч в Arduino
2. Підключіть компоненти
3. Подайте живлення
4. На екрані READY:
   - **OK** → меню
   - **START** → запуск насоса
   - **MENU (довге)** → Wizard

## Основні дії

| Дія | Кнопка | Результат |
|-----|--------|----------|
| Відкрити меню | OK (коротке) | Меню налаштувань |
| Відкрити Wizard | MENU (довге) | Мастер налаштування |
| Запустити насос | START | Запуск/зупинка |
| Регулювати витрату | Потенціометр | Зміна витрати (в RUN) |
| Навігація | Енкодер | Переміщення по меню |

## Швидке налаштування

1. **Wizard** (MENU довге):
   - Матеріал → Діаметр → Рекомендації
   - START → запуск

2. **Меню** (OK коротке):
   - Обертання → вибір пункту
   - OK → редагування
   - MENU → вихід

3. **Калібрування**:
   - Меню → "Calibrate 60s"
   - Зберіть рідину
   - Введіть об'єм в мл

## Налаштування в config.h

```cpp
LCD_I2C_ADDR = 0x27;        // Адреса LCD (0x27 або 0x3F)
ENC_INVERT_DIR = true;      // Інверсія напрямку енкодера
ENC_DETENT_EDGES = 4;       // Переходи на клік (4 для KY-040)
```

## Проблеми?

- **LCD не працює** → перевірте адресу I2C
- **Енкодер не працює** → перевірте D2, D3, A3
- **Насос не запускається** → перевірте D10, D11, D12
