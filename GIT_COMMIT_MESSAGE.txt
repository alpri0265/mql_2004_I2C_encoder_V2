Додано підтримку Arduino Micro для енкодера KY-040

- Додано умовну компіляцію для автоматичного визначення плати
- Виправлено читання бітів PIND для Micro (D2=PD1, D3=PD0)
- Збережено сумісність з Arduino UNO/Nano
- Оновлено коментарі в коді
- Підтверджено сумісність HARD START та HARD DIA керування (використовують digitalRead)

Змінені файли:
- encoder_k040.cpp: додано #if defined(ARDUINO_AVR_MICRO) для readAB_fast()
- encoder_k040.h: оновлено коментарі
- mql_2004_I2C_encoder_V2.ino: додано коментарі про сумісність HARD керування
